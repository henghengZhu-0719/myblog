# AI Agent 技术资讯网站大全 — 国内外优质信息来源

## 引言

AI Agent（智能体/代理）是当前人工智能领域发展最快的技术方向之一。从 OpenAI 的 Agent API 到 Anthropic 的 MCP 协议，从 LangChain 的多 Agent 编排到国内各大厂的智能体平台，Agent 技术几乎每天都在迭代。要持续跟踪这一领域的动态，需要依靠一批高质量的信息来源。本文系统整理了国内外关于 AI Agent 技术的顶级资讯网站，按国际和国内两大部分分类呈现，涵盖新闻聚合、技术博客、社区论坛、研究报告等多种类型。

## 一、国际（英文）网站

### 1.1 每日新闻与资讯聚合站

**AI Agents Directory News**
覆盖 AI Agent 领域最垂直的每日新闻聚合平台，提供每日简报（Daily Briefs）、每周摘要（This Week in AI Agents）和月度归档。内容涵盖 Agent 发布、融资动态和企业采用案例，信息来源透明，且关联到全球最大的 AI Agent 市场目录。地址：https://aiagentsdirectory.com/news [1]

**TechCrunch**
全球顶级科技媒体，设有专门的 AI/Agent 报道板块。深度报道 AI Agent 创业公司融资（如 n8n 获 $180M 融资）、产品发布（如 OpenAI Agent Mode）和行业趋势分析。硅谷最权威的科技媒体之一，Agent 领域的重大新闻几乎都会第一时间覆盖。地址：https://techcrunch.com [2]

**VentureBeat**
领先的科技商业新闻网站，对 AI Agent 企业级应用有深入的报道。设有 AI 专区，定期发布 Agent 行业深度分析文章，对企业级 Agent 部署、AI Agent 创业生态有独特的商业视角分析。地址：https://venturebeat.com/category/ai [3]

**The Decoder**
专注于 AI 和科技深度解读的新闻网站，大量覆盖 AI Agent 领域的最新产品发布、模型更新和技术创新。文章质量高，对 Agent 技术细节有深入解析，不浮于表面。地址：https://the-decoder.com [4]

### 1.2 技术博客

**LangChain Blog**
LangChain 官方博客，发布关于 AI Agent 框架的最新开发进展、最佳实践、技术教程和架构设计深度文章。LangChain 是最流行的开源 Agent 开发框架之一，如果你关注实际的 Agent 技术实现（Agent 架构、工具调用、多 Agent 编排等），这是必读的博客。地址：https://blog.langchain.com [5]

**Anthropic Blog**
Anthropic（Claude 模型开发商）的官方博客，发布关于 Agent 能力、MCP（Model Context Protocol）、Agent2Agent 协议等前沿技术文章。Anthropic 是 Agent 领域的关键驱动者，MCP 协议已成为 Agent 工具集成的事实标准。地址：https://www.anthropic.com/blog [6]

**OpenAI Blog**
OpenAI 官方博客，发布包括 ChatGPT Agent Mode、Frontier 平台、Agents API 等 Agent 相关功能和产品公告。作为 Agent 基础模型的最大提供商，OpenAI 的每项 Agent 功能更新都影响整个行业。地址：https://openai.com/blog [7]

**Glean Work AI Blog**
Glean 的企业 AI 博客，深度探讨企业级 AI Agent（Assistant、Agent Builder、Agent Governance、Agent Orchestration）的架构和落地实践。对企业 Agent 部署、治理和编排有非常实用的技术文章，适合企业技术决策者阅读。地址：https://www.glean.com/blog [8]

### 1.3 邮件通讯（Newsletter）

**TLDR AI**
日更 AI 新闻通讯，拥有 125 万以上订阅者。简明扼要地总结每日 AI 重要新闻，包括 Agent 领域的最新动态。信息密度高、阅读时间短（5 分钟读完），是追踪 Agent 领域每日动态最高效的方式。地址：https://tldr.tech/ai [9]

**The Neuron**
日更 AI Newsletter，55 万多订阅者。以轻松易读的风格覆盖 AI Agent、GPT 等最新进展。内容兼顾技术深度和可读性，适合跨行业团队阅读。地址：https://theneurondaily.com [10]

**AlphaSignal**
专注于 ML 研究与工程的最新进展，18 万多订阅者。技术深度最高，会覆盖 Agent 相关的学术论文、新模型和开源项目，适合希望深入了解 Agent 技术底层原理的 ML 研究人员和工程师。地址：https://alphahunters.substack.com [11]

**Ben's Bites**
面向创始人和投资者的 AI 新闻通讯，12 万多订阅者。重点覆盖 AI Agent 创业公司的融资、产品和市场动态，对 Agent 创业生态有敏锐的洞察。地址：https://bensbites.beehiiv.com [12]

**The Atlas (by GenAI.Works)**
LinkedIn 上最大的 AI 新闻通讯，315 万多订阅者。双周刊形式，覆盖包括 Agent 在内的 AI 行业宏观趋势，社区规模巨大，能第一时间捕捉到 Agent 领域的行业风向变化。地址：https://www.linkedin.com/newsletters/the-atlas-7075090904906149891 [13]

### 1.4 社区与论坛

**Reddit — r/AIAgents**
Reddit 上专注于 AI Agent 讨论的社区。开发者、研究人员和爱好者在这里分享 Agent 构建经验、工具推荐和技术问题解答。每日活跃，能了解一线实践中的痛点、技巧和最新开源项目。地址：https://www.reddit.com/r/AIAgentsForum [14]

**agent.ai**
专业 AI Agent 网络平台，可以构建、发现和激活 AI Agent。拥有 Agent Builder Network 以及 Agent 市场，既是社区也是平台，可以直接与各种 AI Agent 交互，体验最新 Agent 能力。地址：https://agent.ai [1]

**Hugging Face**
最大的开源 AI 社区和模型托管平台，有大量 Agent 相关的模型、框架（如 smolagents）和讨论。每天都有新的 Agent 项目发布，是开源 Agent 生态的核心枢纽。地址：https://huggingface.co [15]
