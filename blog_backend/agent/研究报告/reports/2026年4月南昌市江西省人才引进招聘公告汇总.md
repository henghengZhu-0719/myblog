# 2026年4月南昌市/江西省人才引进招聘公告汇总

## 一、总体概述

2026年4月，江西省及南昌市密集发布了一批人才引进和事业单位招聘公告，涵盖省直事业单位、高校及地市统一招聘等多个层面。招聘规模大、覆盖面广，从博士高层次人才到本科及以上学历均有涉及。以下按发布日期梳理本月发布的核心招聘信息。

---

## 二、南昌市事业单位2026年统一公开招聘（4月29日发布）

南昌市人力资源和社会保障局发布公告，**全市事业单位计划统一招聘755名工作人员**，是本月规模最大的一次招聘。[1]

### 招聘规模

- **总计划**：755名
- **岗位类别**：综合管理类（A类）、社会科学专技类（B类）、自然科学专技类（C类）、医疗卫生类（E类）
- **涵盖单位**：市直事业单位及各县区（开发区、湾里管理局）事业单位

### 关键时间节点

| 环节 | 时间 |
|------|------|
| 网上报名 | 2026年2月4日9:00 — 2月10日17:00 |
| 笔试 | 2026年3月28日（A类）/ 3月29日（B、C、E类） |
| 笔试成绩公布 | 2026年5月中旬 |
| 公告发布日期 | 2026年4月29日（成绩公布前官方转载） |

### 考试科目

- 《职业能力倾向测验》（满分150分）
- 《综合应用能力》（满分150分）
- 总分300分，笔试和面试成绩各占50%

### 待遇与政策

- 事业单位聘用人员符合**南昌新"人才10条"**政策的，可按规定申请补贴
- 公开招聘人员纳入事业编制管理

### 官方渠道

- 南昌市人事考试网：http://www.jxpta.com/jxsrskszx/ncsrsksw/pc/index.html
- 南昌市人力资源和社会保障局官网

## 三、江西省水利厅直属事业单位2026年高层次人才招聘（4月21日发布）

江西省水利厅发布直属事业单位高层次人才招聘公告，面向社会公开招聘博士研究生。[2]

### 招聘计划

- **招聘人数**：博士12名
- **招聘单位**：江西省水利厅直属事业单位
- **岗位性质**：纳入事业编制管理

### 报名与考核

- **公告有效期**：发布之日起至2026年12月31日，岗位计划招满即止
- **流程**：报名 → 资格审查 → 考核 → 体检 → 考察 → 公示 → 聘用
- **考核方式**：由用人单位组织，择优录取

### 待遇保障

1. **纳入事业编制管理**
2. 提供富有竞争力的薪资待遇，适度宽松的科研和工作环境
3. 符合条件者可享受**南昌新"人才10条"政策**
4. 符合条件者可享受省内其他人才引进政策

### 联系方式

- 招聘详情见江西省人力资源和社会保障厅官网或江西省水利厅官网
- 来源：江西人社

## 四、2026年江西省高层次人才专场引才活动（4月18日发布）

江西省教育厅于2026年4月18日发布公告，举办高层次人才专场引才活动。[3]

### 活动信息

- **活动时间**：2026年4月19日（周日）14:00 — 18:00
- **活动地点**：潇湘校区排球场
- **主办单位**：江西省教育厅

### 招聘领域

涵盖理工类、文科类和医学类等多个领域，招聘单位包括中央和国家机关事业单位、地方事业单位以及国际组织，涉及交通运输、医疗卫生、金融经济等热门行业。

### 配套服务

- 设有**理工类就业信息服务群**（群号：727452687）、**文科类群**（群号：161579413）
- 提供线上与线下**简历门诊**服务，帮助求职者优化简历
- 提供线上一对一求职咨询
- 提供校园招聘指南和简历制作系统等资源

## 五、南昌大学2026年人才招聘公告（有效期覆盖4月）

南昌大学面向海内外发布2026年人才招聘公告，全年有效，本月继续接受报名。[4]

### 招聘层次

| 人才层次 | 招聘人数 | 要求概述 |
|----------|---------|---------|
| 杰出人才 | 不限 | 具有很强学术领导力，取得国内外同行公认的开创性成果 |
| 学科领军人才 | 不限 | 在本学科领域取得国内领先或世界先进水平的成果 |
| 学科方向带头人 | 不限 | 在本学科方向取得重要代表性成果 |
| 学科优秀人才 | 不限 | 一般应具有博士后经历（2年及以上且已出站） |
| 其他专业技术岗 | 视岗而定 | 实验员、工程师、馆员、编辑等，博士或正高 |
| 专职辅导员 | 视岗而定 | 需在辅导员岗位连续工作不少于6年 |

### 待遇亮点

- **购房补贴**：19万元
- **人才周转房**：一套
- **配偶安置**：妥善安排配偶工作，不能解决则一次性发放生活补贴
- **子女教育**：幼儿园至高中的一条龙优质教育资源
- **医疗保障**：学校附属三甲医院提供优质医疗保障
- **南昌市人才政策**：博士一次性就业、安家、租房等生活补贴5万元；首次落户南昌发放1000元落户奖励

### 报名方式

- 报名时间：不限（全年开放）
- 报名网址：南昌大学人事处网站人才招聘系统
- 邮箱：rscrsk@ncu.edu.cn
- 联系电话：（0791）83969074（杨老师、陈老师）

## 六、综合分析与建议

### 本月招聘特点

1. **规模大**：南昌市事业单位统一招聘755人，规模创近年新高
2. **层次齐全**：从本科可报的事业单位统考到博士专属的高层次引进，覆盖各学历层次
3. **政策优厚**：南昌新"人才10条"为各类人才提供真金白银的支持，博士可享一次性5万元生活补贴

### 求职建议

- **关注官方渠道**：江西省人力资源和社会保障厅官网（http://www.jxhrss.gov.cn）、南昌市人事考试网、各高校人事处官网是获取权威信息的第一来源
- **博士人才**：可重点关注各高校和省直事业单位的高层次人才引进公告，多数岗位有效期至2026年底，但招满即止，应尽早投递
- **本硕人才**：关注南昌市事业单位统一招聘及后续面试安排，笔试成绩将于5月中旬公布
