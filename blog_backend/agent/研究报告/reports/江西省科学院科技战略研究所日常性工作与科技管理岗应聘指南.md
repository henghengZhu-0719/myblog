# 江西省科学院科技战略研究所日常性工作与科技管理岗应聘指南

## 一、研究所基本定位

江西省科学院科技战略研究所成立于2013年12月17日，是江西省首个集科技战略研究、科技决策咨询、科技信息服务为一体的智库型科研机构，系江西省科学院下属公益一类事业单位 [1][2]。2015年被江西省委省政府确定为江西省重点高端智库，2016年任全国地方科技智库联盟副理事长单位，2017年入选CTTI来源智库 [1]。研究所现有职工56—57人，其中高级职称15人、博士19人、硕士26—28人，青年科研人员占比超过80%，拥有2个国家级平台、3个与中国科学院共建平台及6个省级平台，同时是中国工程科技发展战略江西研究院的运行管理单位 [1][3]。

研究所主要研究方向为**区域创新与科技评价、产业科技与发展战略、知识产权发展战略**三大领域，旨在为江西省科技创新与产业创新发展的宏观决策提供有效支撑 [3]。

## 二、研究所的主要职责与日常性工作

研究所的职责覆盖五个核心板块，以下逐一展开说明。

### 1. 科技战略研究与决策咨询

这是研究所的核心业务，也是日常工作量最大的板块。研究人员围绕省委、省政府的重大决策需求，以及江西省经济和社会发展对科技的需要，开展前瞻性、储备性、针对性研究 [1]。具体工作包括：

- **产业新赛道研究**：跟踪人工智能、新能源、先进材料等前沿技术领域的发展趋势，分析其对江西产业升级的潜在影响，撰写专题研究报告。
- **科技体制机制改革研究**：围绕"十五五"规划要求，研究如何优化创新链、产业链、资金链、人才链深度融合的生态系统，提出政策建议。
- **产业链现代化"1269"行动计划支撑**：为江西省重点产业链的现代化升级提供智力支持，分析产业链短板与突破路径。
- **撰写决策咨询报告（智库专报）**：将研究发现转化为政策建议，报送省领导参阅。截至2023年，研究所累计完成各类报告200余份，获省领导批示400余次 [3]。
- **撰写理论文章**：在《光明日报》《改革内参》《江西日报》等主流媒体发表理论文章，传播研究成果。至今已发表100余篇 [3]。

### 2. 科技查新与知识产权分析评议

研究所面向全省各级政府、行业企业、科研院所、高等院校等社会各界提供科技查新和知识产权分析服务 [2]。日常工作包括：

- **科技查新**：为科研项目立项、成果鉴定、科技奖励申报等提供查新检索服务，出具查新报告。
- **知识产权分析评议**：围绕重点产业和关键技术领域，开展专利导航、知识产权态势分析、知识产权风险评估等工作，为企业和政府提供决策参考。
- **科技情报搜集与分析**：持续跟踪国内外科技发展动态，编译整理科技情报信息，为全省科技工作提供信息支撑 [4]。

### 3. 科技信息服务与图书资源管理

研究所承担为全院职工和科研人员提供图书报刊、数字资源等信息服务的职能 [2]。日常工作包括：

- **数字资源建设与管理**：维护数据库资源（如CNKI、万方、Web of Science等），为科研人员提供文献检索与传递服务。
- **图书报刊管理**：采购、分类、编目图书报刊资料，建设专业馆藏体系。
- **科技信息推送**：根据科研人员需求，定期推送相关领域的最新研究动态和文献信息。

### 4. 信息化建设与运维

研究所为全院信息化建设提供技术支撑及其服务 [2]。日常工作包括：

- 院网站及内部信息系统的维护与更新。
- 网络安全保障与信息化基础设施建设。
- 配合"数字中国"建设要求，推动"人工智能+"在科技管理中的应用。

### 5. 学术交流与平台运营

研究所运营多个高端学术平台和交流机制，日常事务性工作繁重 [3][4]：

- **中国工程科技发展战略江西研究院的运行管理**：组织院士专家为江西重大科技决策提供咨询，协调项目立项与跟踪管理。
- **高端智库论坛活动策划与执行**：如"智库峰会""庐山对话"等品牌活动，至今已举办各类高端论坛活动24个 [3]。
- **对外合作联络**：与中国科学院科技战略咨询研究院、中科院文献情报中心、中国科学技术发展战略研究院等机构保持日常沟通，推动资源共享和协同攻关 [1]。
- **科研项目管理**：研究所累计承担国家级、省级科研项目200余项，争取各类项目经费6000余万元，日常需完成项目申报、中期检查、结题验收等全流程管理 [3]。

## 三、科技管理岗的工作内容分析

根据江西省科学院科技战略研究所2026年人才招聘公告，科技管理岗的岗位要求如下 [4]：

| 项目 | 内容 |
|------|------|
| 学历要求 | 硕士研究生（特别优秀的本科生也可） |
| 专业要求 | 经济学(02)、管理学(12)、法学(03)、交叉学科(14) |
| 年龄 | 30周岁以下 |
| 核心能力要求 | 较强的文字写作、综合表达、沟通能力，具备协调、管理能力 |
| 技能要求 | 熟悉计算机办公软件、网络、统计学等基本知识；能熟练查找和翻译英文资料 |
| 素质要求 | 吃苦耐劳，责任心强，具有较强的分析、解决问题能力和团队协作意识 |

结合研究所的职能框架，**科技管理岗的日常工作**主要包括以下内容：

**（1）科研项目与行政事务管理**
科技管理岗并非直接从事科学研究，而是为科研工作提供管理和支撑。日常需要处理项目申报材料整理、经费使用跟踪、项目进度督促、科研成果统计与归档等事务性工作。需要熟悉科研项目管理的全流程，并具备较强的公文写作能力。

**（2）智库报告的编校与报送**
研究所产出的决策咨询报告、调研报告、智库专报等大量文字材料，需要经过规范化的编校、排版和报送流程。科技管理岗可能负责报告的格式规范、文字校对、印刷制作，以及向省委省政府相关部门的报送协调工作。

**（3）科技查新与情报检索服务**
根据招聘公告中"能熟练查找和翻译英文资料"的要求，科技管理岗可能参与科技查新业务，协助研究人员完成文献检索、资料整理和情报分析等基础性工作。

**（4）学术交流活动的组织协调**
研究所每年举办多场高端论坛、研讨会和学术交流活动。科技管理岗需要参与活动的前期策划、嘉宾联络、会务组织和会后总结等全流程工作。

**（5）信息系统的日常维护**
研究所承担全院信息化建设职能，科技管理岗可能需要参与网站内容更新、数据库管理、办公系统运维等日常技术支持工作。

**（6）对外联络与沟通协调**
研究所与中科院系统、中国工程院、其他省级科学院及政府科技管理部门保持密切合作。科技管理岗可能需要承担日常的对外联络、来访接待、公文流转等沟通协调工作。

## 四、应聘建议

基于以上分析，建议您在应聘准备时重点突出以下方面：

**1. 强化文字功底**
研究所最为看重写作能力，笔试专门考察写作。建议提前准备近期江西省科技政策相关素材（如"1269"行动计划、新质生产力相关论述），练习政策研究报告或调研报告的写作。

**2. 突出沟通协调与管理能力**
面试考察"组织协调能力"，建议准备具体的团队协作、项目管理或活动组织经历案例，用STAR法则呈现。

**3. 展示数据分析和信息检索技能**
公告明确要求熟悉统计学、英文资料查找。如有文献检索、数据分析（Excel/SPSS等）、英文资料翻译等方面的经验，应重点展示。

**4. 了解江西科技发展大局**
建议提前阅读研究所研究人员发表在《江西日报》《光明日报》上的理论文章，熟悉研究所的研究方向和工作语言，在面试中展现对岗位的理解。

**5. 体现服务意识和抗压能力**
公告强调"吃苦耐劳""责任心强"，科技管理岗事务性工作较多，节奏快、任务紧。准备时建议体现你对待繁琐事务的耐心、细致和责任担当。

## 五、结论

江西省科学院科技战略研究所是江西省科技决策领域核心智库，职能覆盖科技战略研究、决策咨询、科技查新、知识产权分析、信息化建设和学术交流等多个维度。科技管理岗虽非直接从事科研，但处于科研支撑和管理协调的核心位置，日常工作涉及项目管、报告编校、情报检索、活动组织、信息系统维护和对外联络等多个方面，需要具备扎实的文字写作能力、组织协调能力和信息处理能力。准备应聘时，建议从文字功底、政策理解、沟通协调和数据分析四个维度重点发力。
