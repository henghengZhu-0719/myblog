# 江西省科学院科技战略研究所综合管理类岗位日常工作内容

## 一、研究所基本概况

江西省科学院科技战略研究所成立于2013年12月17日，是江西省科学院直属的社会公益类科研机构，也是江西省首家集科技战略研究、科技决策咨询、科技信息服务为一体的智库型科研机构[1]。2015年被江西省委省政府确定为江西省重点高端智库，2025年入选中国社会科学评价研究院中国智库综合评价AMI智库入库单位[2]。

研究所现有职工56人，其中高级职称15人、博士19人、硕士26人，拥有2个国家级平台、3个与中国科学院共建平台及6个省级平台[1]。主要研究方向涵盖区域创新与科技评价、产业科技与发展战略、知识产权发展战略三大领域，同时承担中国工程科技发展战略江西研究院的运行管理工作[2]。

## 二、研究所核心职能

根据官方资料，研究所的主要职责分为三大板块[1]：

**决策支撑服务**：围绕省委省政府重大决策及江西省经济和社会发展对科技的需求，为全省各级政府、行业企业、科研院所、高等院校等社会各界提供科技战略研究、科技咨询、科技查新及知识产权分析评议等科技服务，为江西省科技、经济、社会发展的宏观决策和创新驱动发展提供有效支撑。

**内部信息服务**：为全院职工和科研人员提供图书报刊、数字资源等信息服务，保障科研人员的信息获取和知识更新需求。

**信息化建设支撑**：为全院信息化建设提供技术支撑及相关服务。

## 三、综合管理类岗位（科技管理岗）的日常工作内容

结合研究所的职能定位以及2026年公开招聘"科技管理岗"的岗位要求和考核方式[2][3]，综合管理类岗位的日常性工作主要包括以下方面：

### 1. 公文写作与行政文书处理

这是该岗位的核心日常任务之一。招聘公告明确要求"具有较强的文字写作能力"，笔试环节专门考察写作能力[2]。日常工作包括：

- 起草研究所各类公文、报告、请示、函件等行政文书
- 撰写科技战略研究报告、政策建议稿、领导讲话稿等材料
- 整理和管理研究所的各类文件档案，做好收发文登记和归档
- 编制研究所的工作计划、总结、会议纪要等内部管理文书

### 2. 科研项目管理与协调

作为智库型科研机构，研究所每年承担大量国家级、省级科研项目（截至2023年累计承担200余项）[1]，综合管理岗需要参与项目的全流程管理：

- 协助科研项目的申报、立项、中期检查、结题验收等环节的组织工作
- 跟踪项目执行进度，协调项目团队内部的沟通与协作
- 管理项目经费使用，协助编制项目预算和决算报表
- 组织项目评审会、专家咨询会等学术活动的会务工作

### 3. 会议与活动组织协调

招聘公告强调应聘者需具备"协调、管理能力"和"组织协调能力"[2][3]，日常工作中涉及大量会务组织任务：

- 组织学术研讨会、专家论证会、成果发布会等各类学术活动
- 筹备研究所内部行政会议，包括所务会、职工大会、党建会议等
- 协调对外交流活动，包括与中科院科技战略咨询研究院、中国科学技术发展战略研究院等合作单位的对接联络
- 负责中国工程科技发展战略江西研究院运行管理中的日常事务协调

### 4. 科技情报与信息服务

研究所承担科技信息服务职能[1]，综合管理岗需配合开展以下工作：

- 收集、整理国内外科技政策、产业技术动态等信息资料
- 协助科技查新、知识产权分析评议等科技服务业务
- 管理研究所图书报刊和数字资源，为科研人员提供信息检索支持
- 查找和翻译英文科技资料（招聘要求明确提出"能熟练查找和翻译英文资料"）[2]

### 5. 人事与行政事务管理

作为综合管理岗位，日常行政人事事务是不可或缺的部分：

- 协助人员招聘、入职、考核、职称评定等人事管理工作
- 管理职工考勤、差旅、福利等日常行政事务
- 协调研究所内部各团队之间的沟通与协作
- 对接上级单位（江西省科学院）的各项行政工作要求

### 6. 信息化建设相关工作

研究所承担全院信息化建设的技术支撑职责[1]，综合管理岗需参与：

- 协助研究所网站、信息系统等信息化平台的日常维护与管理
- 收集和整理信息化建设需求，配合技术团队推进信息化项目
- 使用计算机办公软件和统计工具进行数据处理与分析（招聘要求"熟悉计算机办公软件、网络、统计学等基本知识"）[2]

### 7. 党建工作与文化宣传

研究所设有"党建与文化"板块[1]，综合管理岗通常也会承担相关工作：

- 协助组织党建学习、主题党日、组织生活会等党建活动
- 撰写党建相关材料和工作报告
- 参与研究所文化建设、新闻宣传等工作

## 四、招聘要求与能力画像

根据2026年招聘公告[2][3]，科技管理岗（综合管理类）对应聘者的核心要求如下：

| 维度 | 具体要求 |
|------|---------|
| 学历专业 | 硕士研究生（特别优秀可放宽至本科）；经济学、管理学、法学、交叉学科 |
| 年龄 | 30周岁以下 |
| 核心能力 | 文字写作能力、综合表达能力、沟通协调能力、管理能力 |
| 专业技能 | 熟悉计算机办公软件、网络、统计学基本知识；能熟练查找和翻译英文资料 |
| 综合素质 | 吃苦耐劳、责任心强、团队协作意识、分析解决问题能力 |

考核方式为笔试（考察写作能力）+ 面试（考察专业知识、表达能力、组织协调能力）[2]。

## 五、求职建议

**突出写作能力**：笔试专门考察写作能力，面试也关注语言表达能力。建议准备时多练习政策类文章写作，关注江西省科技创新相关政策动态。

**强调综合素质**：该岗位虽名为"科技管理岗"，但从日常工作内容来看，行政协调、人事事务、会议组织等综合管理任务占比较大。建议在简历和面试中突出组织协调能力和行政实务经验。

**了解行业背景**：研究所聚焦科技战略与产业创新，面试可能会考察对江西省科技发展战略、产业政策的基本认知，建议提前了解江西省重点产业布局和科技创新政策。

**英文能力佐证**：招聘明确要求英文资料查找和翻译能力，如有相关英语水平证书或翻译经验，可在简历中着重体现。
